Implementação

Opinião do Autor: Processo de adicionar a algo.

Definição (Michaelis): Ato ou efeito de implementar; realização concreta de um plano, projeto ou ideia. Em informática, processo de traduzir um projeto ou especificação em código funcional.

Sinônimos: execução, realização.

HTML

Opinião do Autor: Linguagem de estruturação em websites.

Definição (MDN): HyperText Markup Language (HTML) é a linguagem de marcação padrão para documentos projetados para serem exibidos em um navegador web. Define a estrutura semântica de uma página usando elementos como <p>, <h1>, <div>, etc.

Sinônimos: hipertexto, linguagem de marcação.

JavaScript

Opinião do Autor: Linguagem de interação em websites.

Definição (MDN): JavaScript (JS) é uma linguagem de programação interpretada, de alto nível, que conforma-se à especificação ECMAScript. É usada principalmente para criar interatividade em páginas web, manipulando o DOM e respondendo a eventos.

Sinônimos: JS, linguagem cliente.

Código

Opinião do Autor: Comandos definidos para que uma máquina execute.

Definição (Priberam): Conjunto de símbolos ou sinais convencionados para representar informações; em informática, sequência de instruções em linguagem de programação destinada a ser executada por um computador.

Sinônimos: programação, instruções.

Parâmetro

Opinião do Autor: Uma das coisas necessárias para o funcionamento de um código.

Definição (Houaiss): Cada uma das grandezas que entram numa relação funcional e cujo valor determina o da função; em programação, variável formal declarada na definição de uma função, que recebe um valor ao ser chamada.

Sinônimos: argumento, entrada.

Referente a: Questão 12

Calculado

Opinião do Autor: Algo feito envolvendo números.

Definição (Michaelis): Obtido por meio de cálculo; determinado matematicamente. Em computação, resultado produzido por uma operação aritmética ou lógica executada por um algoritmo.

Sinônimos: computado, determinado.

Verifica

Opinião do Autor: É revisado/olhado.

Definição (Priberam): Examinar com atenção para averiguar a exatidão, a veracidade ou a conformidade com um padrão; em programação, testar uma condição lógica (ex: if) para decidir o fluxo de execução.

Sinônimos: checa, valida.

Tempo

Opinião do Autor: Um conceito universal para definir o período físico das coisas.

Definição (Houaiss): Duração das coisas; sucessão contínua de instantes; em programação, medida de duração (ex: setTimeout) ou ponto no tempo (timestamp em milissegundos desde 1º de janeiro de 1970).

Sinônimos: duração, momento.

Estacionado

Opinião do Autor: Parado

Definição (Michaelis): Que se encontra parado em um lugar; em programação, refere-se a uma variável ou valor que permanece constante durante a execução de um escopo (ex: let/const local).

Sinônimos: fixo, imutável.

Valor

Opinião do Autor: Número ou texto armazenado.

Definição (Priberam): Aquilo que uma grandeza, uma variável ou um símbolo representa; em programação, dado atribuído a uma variável, podendo ser número, string, booleano, objeto, etc.

Sinônimos: dado, conteúdo.

Referente a: Questão 13

Incrementar

Opinião do Autor: Acrescentar a algo.

Definição (Houaiss): Aumentar em uma unidade ou quantidade; em programação, operação que soma um valor (geralmente 1) a uma variável numérica, comumente representada por ++ ou += 1.

Sinônimos: aumentar, somar.

Função

Opinião do Autor: Ordem para executar algo.

Definição (Michaelis): Relação entre grandezas na qual a cada valor de uma corresponde um único valor de outra; em programação, bloco de código nomeado, reutilizável, que pode receber parâmetros e retornar um valor.

Sinônimos: sub-rotina, procedimento.

Lógica

Opinião do Autor: Sentido ou padrão de algo.

Definição (Priberam): Encadeamento coerente e ordenado de raciocínios; em programação, uso de estruturas condicionais (if/else), laços (loops) e operadores para controlar o fluxo de execução.

Sinônimos: raciocínio, coerência.

Agrupar

Opinião do Autor: Juntar, unir.

Definição (Michaelis): Reunir em grupo; em programação, organizar dados relacionados em estruturas como arrays, objetos, funções ou módulos para facilitar manipulação e reutilização.

Sinônimos: reunir, organizar.

ID

Opinião do Autor: Valor de identificação.

Definição (MDN): Atributo global em HTML que define um identificador único (ID) para um elemento, usado para estilização via CSS (#meu-id) ou manipulação via JavaScript (getElementById).

Sinônimos: identificador, chave única.
